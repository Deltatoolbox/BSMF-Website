Gallery Management

How to Add Images

To add new images to the gallery:

  1. Create an images folder in your website directory
  2. Add your image files to the folder
  3. Update the gallery grid in gallery.html
  4. Use the following format for each image:
<div class="gallery-item" data-category="missions">
    <img src="images/your-image.jpg" alt="Description" loading="lazy">
    <div class="gallery-overlay">
        <h3>Image Title</h3>
        <p>Image description</p>
        <span class="image-date">Date</span>
    </div>
</div>

Image Categories

Use these categories for organizing images:

  • missions - Mission screenshots and gameplay
  • team - Team photos and member shots
  • events - Community events and gatherings
  • tactics - Tactical demonstrations and training
